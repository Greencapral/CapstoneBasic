# Агрегатор предложений маркетплейсов OZON и Wildberries

## Как все это запустить
### вариант "local":
* синхронизировать среду командой: **uv sync**
* запустить сервер командой: python manage.py runserver
* как надоест выйти остановить сервер комбинацией **CTRL-BREAK**

### вариант "docker":
* синхронизировать среду командой: **uv sync**
* запустить docker
* собрать и запустить приложение командой: **docker compose up myapp**
* как надоест - прибить все командой: **docker compose down -v**

## Описание проекта
**Веб-сервис** для поиска и сравнения товаров на маркетплейсах OZON и Wildberries по ключевым словам. Проект позволяет пользователям находить товары, сравнивать цены и характеристики в одном месте.

## Цель проекта
Создание удобного инструмента для поиска и сравнения товаров на двух крупнейших маркетплейсах России с возможностью фильтрации и сортировки результатов.

## Основные возможности
### Функционал для пользователей
* Поиск товаров по ключевым словам с поддержкой нескольких запросов* 
* Просмотр результатов в структурированном виде 
* Фильтрация по цене, рейтингу, бренду и другим параметрам 
* Сортировка по релевантности, цене, рейтингу 
* Сравнение товаров в специальном списке 
* Сохранение запросов для быстрого доступа 
* Уведомления о снижении цен

### Данные о товарах
* Название товара 
* Изображение 
* Текущая и старая цена 
* Рейтинг и количество отзывов 
* Ссылка на товар 
* Бренд 
* Продавец 
* Статус наличия

## Технический стек
* Бэкенд: Python 3.10+, Django 4.2+ 
* API: Django REST Framework 
* База данных: PostgreSQL 14+ 
* Контейнеризация: Docker 20.10+, Docker Compose 3.8+ 
* Фоновые задачи (на этапе реализации): Celery 5.3+, Redis 7.0+ 
* Парсинг: ~~feedparser~~, ~~requests~~, ~~Scrapy~~, Selenium 
* Деплой: Docker контейнеры

## Функционал MVP
### Обязательный функционал
* Регистрация и авторизация пользователей 
* Поиск по ключевым словам с базовыми фильтрами 
* Отображение результатов с основными данными 
* Сортировка по цене (возрастание/убывание) 
* Базовые фильтры по площадкам и ценовому диапазону

### Планы развития
* сохранение запросов и история поиска;
* уведомления о снижении цен;
* рейтинги и отзывы;
* сравнение товаров в таблице;
* расширенные фильтры (по бренду, характеристикам и т.д.);
* кеширование данных;
* Адаптивный интерфейс: корректное отображение на ПК и мобильных устройствах;
